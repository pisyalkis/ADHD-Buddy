# ADHD Focus Bot

Telegram-бот-помощник для людей с СДВГ (и всех, кому сложно фокусироваться и не прокрастинировать). Основан на программе тренинга навыков «Mastering Your Adult ADHD» (Safren) в адаптации проекта «СДВГ в квадрате».

Бот не лечит и не заменяет врача — он выступает внешней системой поддержки исполнительных функций: держит структуру дня, приоритезирует задачи и в нужный момент подсказывает конкретный навык вместо общих советов «просто соберись».

## Что бот умеет

- **☀️ Утро** — разминка на 2 минуты → задачи по системе ABC (1×A, 2×B, 3×C) → свободное письмо → благодарность → доброе слово себе. Если вечером были поставлены планы на завтра, бот предлагает оставить их как есть вместо повторного ввода.
- **☕ День** — чек-ин в настроенное время: показывает задачи и, если что-то не идёт (не знаешь с чего начать, перфекционизм, залип в телефоне, сопротивление и т.д.), даёт конкретные техники под ситуацию.
- **🌙 Вечер** — достижения → похвала себя → highlights дня → план A/B/C на завтра.
- **🗂 Карточка дня** — всё, что заполнено за день (утро/день/вечер), собирается в одну карточку с навигацией по датам.
- **🧠 Навык дня** — один из 13 навыков тренинга, меняется каждый день автоматически.
- **🤖 AI-коуч** — свободный текст или быстрые кнопки («не могу начать», «отвлёкся», «прокрастинирую» и т.п.); отвечает через Claude (Anthropic API), если задан ключ.
- **👥 Бадди** — body doubling / accountability buddy: шаблон сообщения, чтобы позвать кого-то поработать рядом.
- **📖 О СДВГ** — короткий гайд: что такое СДВГ, диагностика, группы навыков, почему это работает именно так.
- **🔥 Стрик** — считает подряд идущие дни, когда закрыт вечерний блок.
- **⚙️ Настройки** — время утреннего/дневного/вечернего уведомления задаётся из бота (по умолчанию 09:00 / 13:00 / 21:00), их можно включить/выключить и поменять на своё. Утренние/вечерние уведомления сразу предлагают заполнить нужный блок, а не общее меню.
- **💬 Обратная связь** — можно написать боту идею или проблему прямо из меню; сохраняется в БД и пересылается администратору бота.

Данные хранятся локально в SQLite (`adhd.db`), создаётся автоматически при первом запуске.

## Структура проекта

```
adhd-bot/
├── bot.py             # весь бот: хендлеры, БД, AI, планировщик уведомлений
├── requirements.txt   # зависимости
├── .env.example       # пример переменных окружения
└── .gitignore
```

## Запуск локально

Требуется Python 3.10+.

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

cp .env.example .env
# впиши в .env свой BOT_TOKEN и (по желанию) NOTIFY_USER_ID, ANTHROPIC_KEY
export $(cat .env | xargs)

python3 bot.py
```

Дальше пиши боту `/start` в Telegram.

## Переменные окружения

| Переменная | Обязательна | Описание |
|---|---|---|
| `BOT_TOKEN` | ✅ | Токен от [@BotFather](https://t.me/BotFather) (`/newbot`) |
| `NOTIFY_USER_ID` | нужна для проактивных уведомлений | Твой Telegram ID, узнать у [@userinfobot](https://t.me/userinfobot) |
| `ANTHROPIC_KEY` | ❌ | Ключ [Anthropic API](https://console.anthropic.com/) (`sk-ant-...`). Без него бот работает, но без AI-коуча, утренней мотивации и вечернего анализа |

Время уведомлений (утро/день/вечер) настраивается **внутри бота** через `⚙️ Настройки уведомлений`, хранится в БД и сравнивается с локальным временем сервера — учитывай часовой пояс хостинга (например, задай `TZ=Asia/Tbilisi` в окружении деплоя, если нужно время по Тбилиси).

## Деплой на Railway

1. Создай бота у [@BotFather](https://t.me/BotFather) → `/newbot` → скопируй токен.
2. Залей репозиторий на GitHub (`bot.py`, `requirements.txt`).
3. [railway.app](https://railway.app) → New Project → Deploy from GitHub repo.
4. В Variables добавь `BOT_TOKEN` и (по желанию) `NOTIFY_USER_ID`, `ANTHROPIC_KEY`.
5. Railway сам поставит зависимости из `requirements.txt` и запустит `python bot.py`.

## Стек

- [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot) 20.7
- [APScheduler](https://github.com/agronholm/apscheduler) 3.10 — ежеминутная проверка времени уведомлений
- [anthropic](https://github.com/anthropics/anthropic-sdk-python) 0.18 — AI-коуч и анализ дня (модель `claude-sonnet-4-20250514`)
- SQLite — локальное хранилище (users, diary, tasks, feedback)
